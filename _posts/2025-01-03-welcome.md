Welcome to the Windows 95 Blog! This is a retro-style blog that brings back the nostalgic feeling of using Windows 95. 

## Features

- **Desktop Interface**: Experience the classic Windows 95 desktop with icons, taskbar, and window management
- **File Explorer**: Browse blog posts as if they were files in a file system
- **Command Line**: Access a full command prompt with various commands
- **Search**: Find posts using the built-in search functionality

## Getting Started

1. Double-click on the "File Explorer" icon to browse blog posts
2. Press the backtick key (`) to open the command line interface
3. Use the search bar in File Explorer to find specific posts
4. Double-click on any post to open it in a new window

## Commands Available

In the command prompt, you can use:
- `help` - Show available commands
- `ls` - List all posts
- `cat <title>` - Display post content
- `open <title>` - Open post in new window
- `search <term>` - Search posts
- `echo <text>` - Display text

Enjoy your retro computing experience!
