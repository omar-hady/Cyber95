# Welcome to Windows 95 Blog

This is a retro-style blog built with Jekyll that mimics the Windows 95 desktop experience.

## Features

- **Desktop Interface**: Full Windows 95 desktop with icons, taskbar, and window management
- **File Explorer**: Browse blog posts as files in a file system interface
- **Command Line**: Access a CMD window with various commands
- **Search**: Find posts using the built-in search functionality
- **Responsive**: Works on desktop and mobile devices

## Getting Started

Double-click on the "File Explorer" icon to browse blog posts, or press the backtick key (`) to open the command line interface.

Enjoy your retro computing experience!
